# odin-recipes

## Introduction
An HTML5 project featuring three simple Filipino cuisine recipes:
- Utan Bisaya
- Pork Adobo
- Tortang Talong 

## Skills Demonstrated
Command Line:
- creating files and directories using touch and mkdir
- moving files and directories using mv with wildcards
- renaming files using mv
- moving through directories using cd and cd ..
- deleting files and directories with rm -i or rm -r -i

Git and Github:
- creating and cloning repos on local machine
- using git add . to add the changes to the staging area
- using git commit to save current state on local repo
- using git push origin main to make changes on main branch
- writing commit messages

HTML:
- HTML Boilerplate shortcuts using ! + enter
- using Elements and tags to structure content based on standard
- using headers and paragraphs for heading and paragraph contents
- using anchors and img for relative links and images
- using ordered and unordered lists for content that are list-like
- 

Other:
- Reading docs
- File organization

## Comments
As a person who uses tech alot but not create them, I though it required some
esoteric knowledge to make a webpage. Thanks to The Odin Project curriculum,
I was able to make one with this particular HTML5 page project.

I have to admit while making this project I was rereading previous lessons on 
multiple occasions though the curriculum does encourage it than memorizing
the lessons. Perhaps this is how the entire curriculum is meant to be done, 
a project driven learning.

I felt like I could have used the strong and italicized tags more to make the
content readable. But then again I get the basic lesson already and thats what Odin
Project is trying to teach, learn enough then move.
